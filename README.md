# Entify-Historical-NER-NLP-Project

Entify: RoBERTa-Based Named Entity Recognition System
This repository contains the source code for the "Entify" project, an NLP system designed to perform Named Entity Recognition (NER) on historical texts using Transformer-based architectures.

Repository Contents
app.py: The Streamlit web application code for the Entify user interface.

notebooks/: Contains the experiment files for model training (BERT, RoBERTa, XLM-RoBERTa, and BiLSTM-CRF).

Project Overview
This project aims to automate the extraction of historical information (PER, LOC, ORG, MISC) from documents. Our findings show that Standard RoBERTa achieves the highest F1-score (0.95) compared to other architectures.
